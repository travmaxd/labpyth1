# Лабораторная работа №1 по теории алгоритмов

Этот проект содержит решения различных учебных задач по программированию на Python, демонстрирующих работу со строками, списками, словарями, множествами и базовыми вычислениями.

## Быстрый старт

### Предварительные требования
- Python 3.6 или выше

### Установка и запуск

### 00. Distance - Расстояния между городами
#### Задача: Вычислить расстояния между городами по координатам
Формула: √((x₁ - x₂)² + (y₁ - y₂)²)
Особенности:
- Используется словарь с координатами городов
- Рассчитываются попарные расстояния
- Результаты сохраняются в словарь с ключами "Город1-Город2"

### 01. Circle - Работа с кругом
#### Задача: Вычислить площадь круга и проверить принадлежность точек
#### Формулы:
- Площадь: S = π × r²
- Расстояние до точки: √(x² + y²)
Параметры:
- Радиус: 42
- π: 3.1415926
- Точки: (23, 34) и (30, 30)

### 02. Operations - Математические головоломки
#### Задача: Расставить операции между числами 1 2 3 4 5 для получения 25
Решение: ((1 + 2) × 3 - 4) × 5 = 25

Этапы:
<div style="margin-left: 40px;">

   (1 + 2) = 3
    3 × 3 = 9
    9 - 4 = 5
    5 × 5 = 25
</div>

### 03. Favorite Movies - Работа со строками
#### Задача: Извлечь фильмы из строки используя только срезы
Методы: Используется только индексация строки
<div style="margin-left: 40px;">

Первый фильм: [:10]
Последний: [37:]
Второй: [12:26]
Второй с конца: [29:35]
</div>

### 04. My Family - Работа со списками
#### Задача: Создать структуру данных семьи и вычислить параметры
Пример:
<div style="margin-left: 20px;">
Пример:

    my_family_height =[
    ['Мама', 167],
    ['Папа', 180],
    ['Оля', 165]
    ]
</div>
Вычисления: Рост папы и общий рост семьи

### 05. Zoo - Операции со списками
Задачи:
- Добавить медведя между львом и кенгуру
- Добавить птиц в конец
- Удалить слона
- Найти позиции льва и жаворонка
Методы: insert(), extend(), remove(), index()

### 06. Songs List - Работа со временем
#### Задача: Вычислить общее время звучания песен
Для списка: 'Halo', 'Enjoy the Silence', 'Clean'
Для словаря: 'Sweetest Perfection', 'Policy of Truth', 'Blue Dress'
Особенности: Использование функции round() для контроля точности

### 07. Secret - Расшифровка сообщения
Алгоритм расшифровки:
1 слово: 4-я буква 1-й строки
2 слово: буквы 10–13 2-й строки
3 слово: буквы 6–15 через одну 3-й строки
4 слово: буквы 8–13 в обратном порядке 4-й строки
5 слово: буквы 17–21 в обратном порядке 5-й строки

### 08. Garden - Операции с множествами
Задачи:
- Все уникальные цветы: | (объединение)
- Общие цветы: & (пересечение)
- Только в саду: - (разность)
- Только на лугу: - (разность)

### 09. Shopping - Анализ цен
#### Задача: Найти 2 магазина с минимальными ценами для каждого товара
Алгоритм:
- Реорганизация данных по товарам
- Поиск минимальных цен
- Формирование словаря результатов

### 10. Store - Расчет стоимости товаров
Задача: Рассчитать количество и стоимость товаров без циклов
Товары: Лампа, Стол, Диван, Стул

- **garden()** - проверка операций с множествами цветов
- **family()** - проверка расчетов с данными семьи
- **operations()** - проверка математических выражений
- **secret()** - проверка расшифровки сообщения
- **shops()** - проверка структуры данных магазинов
- **songs()** - проверка расчета времени звучания
- **store()** - проверка расчетов стоимости товаров
- **zooF()** - проверка операций со списком животных

### Запуск тестов 

Установка зависимостей
pip install pytest

Запуск всех тестов
pytest test/test_my_functions.py -v

Запуск конкретного теста
pytest test/test_my_functions.py::test_circle -v

Запуск с подробным выводом
pytest test/test_my_functions.py -v --tb=short

Что проверяют тесты
Каждый тест проверяет:
Корректность типов данных
Правильность вычислений
Структуру данных
Логику работы
Граничные случаи
